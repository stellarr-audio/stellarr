# Stellarr User Manual

Welcome to the Stellarr user manual. Use the sidebar to navigate, or start with the [Introduction](01-introduction.md).

New here? Check out the [Quick Start](02-quick-start.md) guide to get up and running in under five minutes.

## Viewing This Manual

To browse the manual locally with navigation and search:

```bash
make docs
```

Then open [http://localhost:3001](http://localhost:3001) in your browser.
